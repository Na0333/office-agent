# 邮件发送功能使用指南

## 概述

Office Agent 现已内置独立的邮件发送功能，无需依赖 AstrBot 的 mailer 插件。该功能支持配置 SMTP 服务器，可以发送普通邮件、通知邮件以及支持附件和抄送功能。

## 功能特性

- ✅ **SMTP 支持**：支持隐式 TLS（端口 465）和 STARTTLS（端口 587）两种连接方式
- ✅ **灵活配置**：通过环境变量配置，支持 QQ 邮箱、Gmail、企业邮箱等
- ✅ **多收件人**：支持单个或多个收件人、抄送和密送
- ✅ **附件支持**：支持 PDF、Word、Excel 等文档附件
- ✅ **HTML 格式**：支持 HTML 格式邮件，提升邮件展示效果
- ✅ **通知邮件**：专门的通知邮件格式，自动处理待办事项的邮件转发
- ✅ **配置验证**：提供 SMTP 连接验证工具，快速诊断配置问题

## 配置指南

### 1. 基础配置

编辑项目根目录的 `.env` 文件，添加以下配置：

```env
# SMTP Email Configuration
SMTP_HOST=smtp.qq.com
SMTP_PORT=465
SMTP_USERNAME=your-email@qq.com
SMTP_PASSWORD=your-smtp-password
SMTP_TIMEOUT=30
SMTP_USE_TLS=true
SMTP_USE_STARTTLS=false
```

### 2. 常见服务商配置

#### QQ 邮箱（推荐）

```env
SMTP_HOST=smtp.qq.com
SMTP_PORT=465
SMTP_USERNAME=your-email@qq.com
SMTP_PASSWORD=your-smtp-password
SMTP_USE_TLS=true
SMTP_USE_STARTTLS=false
```

**获取授权码：**
1. 登录 QQ 邮箱
2. 进入 "设置" → "账户" → "POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
3. 开启 "POP3/SMTP服务"
4. 获取授权码（授权码就是密码）

#### Gmail

```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=your-app-password
SMTP_USE_TLS=false
SMTP_USE_STARTTLS=true
```

**获取应用密码：**
1. 启用两步验证
2. 生成应用专用密码

#### 企业邮箱（Outlook）

```env
SMTP_HOST=smtp.office365.com
SMTP_PORT=587
SMTP_USERNAME=your-email@company.com
SMTP_PASSWORD=your-password
SMTP_USE_TLS=false
SMTP_USE_STARTTLS=true
```

#### 腾讯企业邮

```env
SMTP_HOST=smtp.exmail.qq.com
SMTP_PORT=465
SMTP_USERNAME=your-email@company-domain.com
SMTP_PASSWORD=your-password
SMTP_USE_TLS=true
SMTP_USE_STARTTLS=false
```

## 使用方式

### 方式一：Agent 流程中自动发送

在规划邮件发送任务时，Agent 会自动调用邮件服务：

```python
# 用户指令
/office 请整理会议纪要成待办，并发送通知邮件到 zhangsan@company.com

# Agent 自动执行流程：
# 1. 提取待办项
# 2. 生成通知
# 3. 发送邮件通知
```

### 方式二：直接调用邮件工具

在代码中直接调用 MCP 工具：

```python
from app.mcp_client.client import MCPToolClient

client = MCPToolClient()

# 发送普通邮件
result = await client.call("send_email", {
    "to_addresses": "recipient@example.com",
    "subject": "会议纪要",
    "body": "以下是本次会议的纪要...",
    "html": False,
    "cc": "cc@example.com",
    "bcc": ""
})

# 发送通知邮件
result = await client.call("send_email_notification", {
    "to_address": "recipient@example.com",
    "notice_content": "以下是今日待办事项...",
    "email_subject": "待办事项通知"
})
```

### 方式三：使用 EmailService 服务

```python
from app.services.email_service import EmailService

service = EmailService()

# 发送邮件
result = await service.send_email(
    to_addresses="recipient@example.com",
    subject="Test",
    body="Test email body",
    html=False,
    cc=["cc@example.com"],
    bcc=["bcc@example.com"]
)

# 验证配置
validation = await service.validate_smtp_config()
print(f"SMTP 有效: {validation['valid']}")
```

## API 接口

### 发送邮件

**工具名称**：`send_email`

**参数**：
- `to_addresses` (str): 收件人地址，支持逗号分隔的多个地址
- `subject` (str): 邮件主题
- `body` (str): 邮件内容
- `html` (bool, optional): 是否为 HTML 格式，默认 False
- `cc` (str, optional): 抄送地址，逗号分隔
- `bcc` (str, optional): 密送地址，逗号分隔

**返回值**：
```json
{
  "success": true,
  "message": "Email sent to recipient@example.com",
  "message_id": "unique-message-id",
  "error": null
}
```

### 发送通知邮件

**工具名称**：`send_email_notification`

**参数**：
- `to_address` (str): 收件人地址
- `notice_content` (str): 通知内容
- `email_subject` (str, optional): 邮件主题，默认 "办公协同通知"

**返回值**：
```json
{
  "success": true,
  "message": "Email sent to recipient@example.com",
  "message_id": "unique-message-id",
  "error": null
}
```

### 验证配置

**工具名称**：`validate_email_config`

**参数**：无

**返回值**：
```json
{
  "valid": true,
  "message": "SMTP connection successful: smtp.qq.com:465",
  "error": null
}
```

## 故障排除

### 问题：连接超时

**原因**：SMTP 服务器无法访问或防火墙限制

**解决**：
1. 检查 SMTP_HOST 和 SMTP_PORT 是否正确
2. 检查网络连接
3. 检查防火墙设置

### 问题：认证失败

**原因**：用户名或密码错误

**解决**：
1. 确认 SMTP_USERNAME 和 SMTP_PASSWORD 正确
2. 对于 QQ 邮箱，确保使用授权码而非密码
3. 检查是否启用了服务

### 问题：TLS/STARTTLS 错误

**原因**：连接方式配置错误

**解决**：
- 端口 465：使用 `SMTP_USE_TLS=true`，`SMTP_USE_STARTTLS=false`
- 端口 587：使用 `SMTP_USE_TLS=false`，`SMTP_USE_STARTTLS=true`

### 问题：邮件被拒收

**原因**：收件人过多、发送频率过高或被列入黑名单

**解决**：
1. 减少一次发送的收件人数量
2. 降低发送频率
3. 联系邮件服务商申请解封

## 安全建议

1. **不要在代码中硬编码密码**，始终使用环境变量
2. **使用授权码而非账户密码**（QQ 邮箱强烈推荐）
3. **限制发件人地址**，避免被用于垃圾邮件
4. **定期轮换密码**，增强账户安全性
5. **监控邮件发送日志**，及时发现异常
