## 🚀 部署步骤

1. **环境配置**
   ```bash
   # 编辑 .env 或创建新的 .env 文件
   cp .env.example .env
   # 填入 SMTP_HOST, SMTP_PORT, SMTP_USERNAME, SMTP_PASSWORD
   ```

2. **验证配置**
   ```bash
   python test_email_quick.py
   ```

3. **运行测试**
   ```bash
   pytest tests/test_email_service.py -v
   pytest tests/test_email_integration.py -v
   ```

4. **启动服务**
   ```bash
   python -m app.main
   ```

5. **测试功能**
   ```bash
   # 通过 API 或 AstrBot 与 Agent 交互
   # 发送包含邮件的请求
   ```
