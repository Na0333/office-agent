## 📋 工具总览

## 📧 配置向导（最常用）

```bash
python setup_email.py
```

**5 步完成配置：**

### 步骤 1：选择邮箱提供商

```
请选择你的邮箱提供商：

1. QQ 邮箱
2. Gmail  
3. Outlook / Hotmail
4. 腾讯企业邮
5. 其他 SMTP 服务
```

支持直接输入邮箱域名自动识别。

### 步骤 2：输入邮箱地址

```
请输入你的邮箱地址: your-email@qq.com
```

### 步骤 3：输入授权码

```
📖 如何获取授权码？
   1. 登录 QQ 邮箱
   2. 进入 '设置' → '账户'
   3. 找到 'POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务'
   4. 点击 '生成授权码'

请输入授权码或密码: xxxxxxxx
```

**自动化流程：**
- 🔍 自动显示获取授权码的步骤
- 📝 粘贴授权码即可
- ✅ 验证长度和格式

### 步骤 4：配置 SMTP 参数

根据选择的提供商自动预配置（无需手动操作）：

```
已为 QQ 邮箱 预设配置参数：

✓ SMTP 服务器: smtp.qq.com
✓ SMTP 端口: 465
✓ 使用 TLS: true
✓ 使用 STARTTLS: false
```

或选择自定义配置。

### 步骤 5：确认并保存

```
📋 配置摘要：

邮箱地址: your-email@qq.com
SMTP 服务器: smtp.qq.com
SMTP 端口: 465
使用 TLS: true
使用 STARTTLS: false

确认配置正确? (y/n) [y]:
```

**自动功能：**
- ✅ 自动保存到 `.env`
- ✅ 保留现有配置
- ✅ 提示测试邮件

## 🔍 诊断工具（遇到问题时使用）

```bash

```

**检查项目：**

```
1️⃣  检查配置文件

2️⃣  检查 SMTP 配置参数

   ✅ SMTP_PORT: 465
   ✅ SMTP_USERNAME: your-**@qq.com
   ✅ SMTP_PASSWORD: ********

3️⃣  检查 SMTP 连接
   ✅ SMTP 连接成功: smtp.qq.com:465

4️⃣  检查邮件发送功能
   ✅ 邮件已发送

5️⃣  检查 MCP 工具集成
   ✅ 工具 send_email 已注册
   ✅ 工具 send_email_notification 已注册
   ✅ 工具 validate_email_config 已注册

6️⃣  检查 Agent 集成
   ✅ 邮件意图识别正常
```

**问题诊断：**

如果遇到问题，工具会自动：
1. 🔍 定位问题位置
2. 📋 显示具体错误
3. 💡 提供修复建议

**示例：**

```
❌ [ERROR] SMTP 连接失败: [Errno 110] Connection timed out
   💡 建议: 检查服务器地址、端口和网络连接
```

## 💡 使用技巧

### 技巧 1：快速配置

如果你知道自己的 SMTP 信息，配置向导只需 2-3 分钟：

1. 选择提供商 (15 秒)
2. 输入邮箱地址 (15 秒)
3. 输入授权码 (30 秒)
4. 确认 (10 秒)
5. 自动保存完成 (5 秒)

**总时间：< 2 分钟** ⚡

### 技巧 2：多个邮箱

如果要配置多个邮箱：

```bash
# 第一次配置
python setup_email.py
# 配置邮箱 A，保存到 .env

# 如果要配置邮箱 B，重新运行
python setup_email.py
# 会问是否覆盖，选择 "y" 即可
```

### 技巧 3：快速验证

配置完成后，快速验证一下：

```bash
python diagnose_email.py
# 如果所有项都是绿色 ✅，配置成功！
```

### 技巧 4：获取授权码

如果忘记如何获取授权码，配置向导会自动显示步骤：

```
📖 如何获取授权码？
   1. 登录 QQ 邮箱
   2. 进入 '设置' → '账户'
   ...
```

不用查文档，直接跟着步骤做即可。

## ❓ 常见问题

**Q: 我应该用哪个工具？**  
A: 新用户用 `init.py`，之后根据需要用其他工具。

**Q: 我不知道我的邮箱类型？**  
A: 可以直接输入邮箱地址，系统会自动识别（如 @qq.com）。

**Q: 我找不到授权码？**  
A: 配置工具会自动显示步骤，跟着做即可。

**Q: 配置好了怎么测试？**  
A: 运行 `python diagnose_email.py` 或 `python test_email_quick.py`。

**Q: 配置文件在哪？**  
A: 在项目根目录的 `.env` 文件中。
